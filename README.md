# mothers-day-carrie

A handwritten-style Mother's Day letter from Mike & Kelsey Van Doorn to Carrie, rendered as a parchment document with calligraphic fonts, hand-drawn SVG sketches, and a wax seal.

## Local dev

Open `index.html` directly in a browser — no build step needed.
